Asking About Python

Ask Mary about Python

This is a simple little project for my participation on the MLH Summer League. Below is Mary and she can help you with some basic information about Python.

  • Ask Mary basic questions about Python to help you
  • For example ask her "What is Python ?" or "What is a class ?"
  • Just to inform you that Mary is still learning so dont expect to answer everything about Python but she is getting close..
  • But first say a Hello..

The project

This project is my submission for MLH Summer League Hackathon. In this simple project i decided to make a simple example with an educational goal to showcase what you can do with bots. I used Dialogflow from Google Cloud to get my first results. You can read more in general about Google Cloud services at Google Cloud Official web site.And for the Dialogflow service at Dialogflow.
